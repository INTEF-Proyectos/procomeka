# Evaluación de Cumplimiento RGPD/LOPDGDD

## 1. Identificación del Tratamiento
- **Responsable**: INTEF/MEFP.
- **Finalidad**: Gestión de la plataforma de recursos educativos abiertos, perfiles de autoría y participación comunitaria.
- **Interesados**: Personal docente y usuarios registrados.

## 2. Categoría de Datos
- **Datos Identificativos**: Nombre, apellidos, correo electrónico.
- **Datos de Perfil**: Rol (autor, curador, administrador), biografía, imagen de perfil.
- **Datos de Actividad**: Historial de recursos publicados, comentarios, valoraciones, favoritos.
- **Datos de Conexión**: Dirección IP (para auditoría de seguridad).

## 3. Principios de Protección de Datos
- **Minimización**: Solo se recogen los datos estrictamente necesarios para la creación y gestión del perfil.
- **Integridad y Confidencialidad**: Los datos personales están protegidos por medidas técnicas (RBAC, TLS).
- **Limitación de la Finalidad**: Los datos no se usan para fines ajenos a la plataforma Procomeka.
- **Limitación del Plazo de Conservación**: Los datos se conservan mientras la cuenta esté activa o sea necesario para la atribución de autoría de recursos públicos.

## 4. Derechos de los Interesados (ARPOL)
- El sistema permite el acceso a los datos propios a través del dashboard.
- Se facilita la rectificación de los datos de perfil.
- Se implementará un mecanismo de supresión de cuenta respetando la integridad del catálogo de recursos públicos (atribución).

## 5. Medidas Técnicas de Seguridad
- Registro de acceso a datos personales (Middleware de auditoría).
- Separación de privilegios para la administración de usuarios (Roles).
- Cifrado en tránsito (TLS).
